## 精简原则

- 文档和表格是项目文件中的两类文件，不作为两个并列主产品；项目文件负责统一增删改查、点击打开、权限和日志。
- 成员级任务进度是任务模型的一部分，不做独立学习系统、工时系统或绩效系统。
- 文件提交是任务资料的一部分，不做重型网盘。
- 系统支撑能力是后台能力，默认不进入普通用户主导航。
- 个性化只在头像菜单的用户卡片中出现，不进入主导航和主工作区。
- 用户信息设置从用户卡片进入，只维护头像、卡片背景、个性签名和密码。
- 第一版只做项目闭环必须能力，表格转储、高保真 Office 转换、复杂规则编排、复杂学习系统和工时绩效后置。

| 资源对象 | 新增 | 查看 | 修改 | 删除 / 停用 | 备注 |

| 项目 | 创建项目 | 项目列表、项目详情 | 编辑项目、项目设置 | 删除/关闭、归档、恢复 | 删除建议软删除，归档可恢复。 |
| 任务 | 创建任务、复制任务 | 任务列表、任务详情、看板、甘特 | 编辑任务、状态拖拽、修改负责人/时间/依赖/优先级 | 删除、归档、恢复 | 状态变更写时间线。 |
| 成员进度 | 新增成员进度 | 成员进度详情、个人甘特 | 编辑成员计划、上报完成/延期/阻塞/放弃 | 移除成员进度 | 移除需保留历史日志。 |
| 项目成员 | 邀请成员 | 成员列表、在线状态 | 编辑成员角色 | 移除成员 | 项目创建者拥有完整管理权。 |
| 验收项 | 新增验收项 | 验收清单、验收报告 | 编辑验收项、标记通过/需修改/驳回 | 删除验收项 | 验收结果进入归档包。 |
| 项目文件 | 新建、上传、导入 | 文件列表、预览、版本 | 重命名、编辑、移动、回滚 | 删除、恢复 | Word/表格/附件/提交物统一管理。 |
| 任务提交物 | 上传成果 | 收集箱、提交状态 | 标记需修改、已验收、重新提交 | 删除/撤回提交 | 权限控制是否可看/下载别人提交物。 |
| 消息规则 | 新建规则、复制规则 | 规则列表、规则详情 | 编辑规则、提醒对象、启停 | 删除规则 | 发送日志只读，失败可重试。 |
| 通知渠道 | 新增渠道配置 | 渠道状态、检测结果 | 修改 Key、启用/禁用 | 删除渠道配置 | Key 加密保存，不显示明文。 |
| 用户 | 创建/邀请用户、新增 IP 白名单条目 | 用户列表、用户详情、白名单拦截记录、会话列表 | 编辑用户、启用/禁用、重置密码、开启/关闭 IP 白名单、编辑白名单条目 | 删除/停用用户、删除白名单条目、吊销会话 | 删除建议停用优先；白名单变更写安全审计。 |
| 角色模板 | 新建/复制角色 | 查看角色模板 | 编辑角色权限 | 删除自定义角色 | 内置角色不可删除，只能复制。 |
| 权限范围 | 新增项目权限范围 | 权限矩阵 | 修改进度/文件/下载/验收权限 | 删除自定义范围 | 权限变更写审计。 |
| 用户资料 | 上传头像/背景 | 用户卡片、资料页 | 修改头像、签名、密码、背景 | 删除背景、重置资料 | 从头像卡片进入。 |
| 个性化皮肤 | 上传自定义背景 | 查看当前皮肤 | 修改皮肤、模糊度、强调色 | 删除背景、重置默认皮肤 | 不占主导航。 |

不套 CRUD 的只读/派生对象：仪表盘指标、时间线、审计日志、服务器监控、发送日志、甘特筛选结果。它们只提供查看、筛选、跳转、重试或保存视图偏好。
