# LightTask v11 文档索引

v11 是在 v10 成员进度版基础上的“协作闭环增强版”。本版本不新增学习系统、重型网盘、工时系统或绩效系统，而是把第二批真实协作所需能力继续挂到现有任务、甘特、资料、消息、权限和验收里：甘特图高级筛选、定向提醒、验收统计报告和权限细分。

## 当前产品一句话

LightTask v11 是一个轻量级多人项目协同系统，以项目主线、成员进度、资料提交、计划对比、消息提醒和验收复盘为核心，帮助小团队清楚知道“谁在做什么、进度快慢多少、文件交了没有、下一步该提醒谁”。

## v11 核心变化

- 保留 v9 的全局框架、登录框、头像个性化、项目工作台、项目文件、消息同步和管理中心。
- 新增成员级任务进度：一个任务可拆到多个成员，每个成员有自己的计划、实际完成、状态、提交物和说明。
- 新增计划基线、当前计划、实际进度三层时间，用于计算快/慢预期多少天。
- 仪表盘新增“我的进度”视角，展示今日、本周、本月完成情况、预计完成日期和快慢天数。
- 项目工作台新增任务上报动作：完成、延期、阻塞、放弃、休息一下、继续下一任务、上传成果、提醒创建者。
- 项目文件合并在线文档、在线表格、附件和文件收集箱；Word、表格、附件点一下即可在同一工作区打开，并支持新建、上传、重命名、删除、导出。
- 甘特图升级为高级筛选：项目主线、原计划对比、当前进度、指定成员、团队完成节点、只看延期、只看我负责。
- 消息同步升级为定向提醒：全部成员、只提醒负责人、只提醒创建者、自定义成员、不提醒。
- 验收归档升级为验收统计报告：成员计划/实际完成、快慢天数、延期次数、放弃次数、阻塞次数、提交文件数量和验收状态。
- 权限模型落地细分：进度可见、文件可见、文件下载、提交物验收四类权限独立配置。

## 第一批已加入核心版

1. 成员级任务进度。
2. 计划基线 / 当前计划 / 实际进度。
3. 个人进度仪表盘。
4. 任务上报动作。
5. 项目文件统一管理：Word、表格、附件、任务提交物和项目文件收集箱。

## 第二批已加入 v11

1. 甘特图高级筛选：默认看项目主线，需要时再展开成员、延期、我负责和团队节点。
2. 定向提醒：任务下发、延期、完成、验收、文件未交等场景可选择提醒对象。
3. 验收统计报告：项目完成后自动汇总成员交付表现和提交物验收状态，用于归档和复盘。
4. 权限细分：把进度可见、文件可见、文件下载和验收权限拆开，避免“能看进度”误等于“能看文件”。
