# 在线文档与表格 v6 操作界面设计说明

日期：2026-06-01

## 1. 设计目标

本轮重新设计在线文档和表格的操作界面，参考石墨文档类产品的核心操作逻辑：文件空间、实时编辑、多人协作、评论锚点、版本回溯、导入导出和权限分享都在同一个编辑工作台完成。

## 2. 页面结构

采用三栏结构：

- 左侧文件空间：项目空间、文件搜索、文件树、文档/表格切换、Office/WPS 导入入口。
- 中间编辑区域：顶部标题栏、分享和协作成员、文档/表格模式切换、格式工具栏、正文或表格画布。
- 右侧协作面板：评论、版本、成员、导入导出兼容信息。

## 3. 在线文档操作逻辑

在线文档模式包含：

- 文档标题栏：显示路径、标题、保存状态、协作者、评论、分享。
- 富文本工具栏：正文样式、字号、加粗、斜体、下划线、清单、引用、表格、图片、@ 提及、历史、导出。
- 大纲面板：可快速跳转验收范围、上线检查项、风险与待办、附件与表格。
- 纸张画布：承载正文、评论锚点、远程光标和检查表。
- 协作侧栏：评论处理、版本记录、导入导出兼容报告。

用户路径：

1. 从左侧文件树选择文档。
2. 在中间画布编辑正文。
3. 使用评论锚点讨论具体段落。
4. 右侧查看版本、回溯历史、处理评论。
5. 通过分享按钮邀请成员或调整权限。

## 4. 在线表格操作逻辑

在线表格模式包含：

- 表格标题栏：显示当前表格、保存状态、正在编辑的单元格和协作者。
- 表格工具栏：撤销、重做、格式刷、货币、百分比、筛选、冻结、函数、图表、导入、导出。
- 公式栏：显示当前单元格与公式内容。
- 表格网格：列头、行号、选中单元格、多人编辑标记、状态标签。
- 底部工作表标签：预算排期、风险清单、成员工时。
- 协作侧栏：单元格评论、版本记录、Excel/WPS 兼容报告。

用户路径：

1. 从文件树打开表格。
2. 通过公式栏和网格进行编辑。
3. 使用筛选、冻结、函数和图表完成表格操作。
4. 右侧处理单元格评论和版本差异。
5. 导入或导出 xlsx，并查看兼容性报告。

## 5. 协作状态

界面明确展示：

- 已同步 / 正在同步。
- 正在编辑的成员。
- 远程光标和选区。
- 评论锚点。
- 版本记录。
- 导入导出兼容率。
