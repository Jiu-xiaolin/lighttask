# 在线文档 Markdown 编辑器 v7 设计说明

## 设计目标

本版将在线文档调整为类似石墨文档/知识库的操作逻辑：左侧目录定位，中间 Markdown 所见即所得编辑，右侧自动大纲。重点服务长文档、技术文档、项目验收说明、会议纪要和知识库沉淀。

## 页面结构

- 左侧知识库树：展示空间名称、权限状态、目录层级和当前选中文档；支持新建文档、导入 Markdown / Office / HTML。
- 顶部文档栏：显示路径、标题、同步状态、收藏、评论、成员和分享入口。
- Markdown 工具栏：功能按钮使用图标表达，保留 `aria-label` 与悬停说明；首屏仅展示高频操作，低频格式进入“更多”菜单，减少编辑区顶部拥挤感。
- 中央编辑区：默认所见即所得，同时提供 Markdown 源码和预览模式；块级内容左侧保留插入按钮和 Markdown 标记。
- 右侧大纲：根据 H1/H2/H3 自动生成目录，并显示 Markdown 能力、版本与多人协同状态。

## Markdown 能力

- 支持标题、正文、引用、代码块、任务清单、表格、图片、链接和图示。
- 支持 `/` 快捷插入块，适合快速创建项目验收模板、接口说明、漏洞复现步骤等内容；工具栏提供插入、撤销/重做、格式刷、标题、加粗、斜体、下划线、代码、列表、待办、表格、图片和更多操作。
- 代码块保留语言标记，导出 Markdown / docx / pdf 时保留结构。
- 表格内容可协同编辑，并能同步到项目时间线中的文档变更日志。

## 多人协同与版本

- 显示在线成员、远程光标、正在输入状态和自动保存状态。
- 每次标题、表格、图片、代码块和正文修改都会进入版本记录。
- 版本回溯、差异对比、评论锚点会以卡片形式同步到项目工作台右侧变更时间线。

## 导入导出兼容

- 导入：`.md`、`.docx`、`.html`、`.txt`。
- 导出：`.md`、`.docx`、`.pdf`、`.html`。
- Office / WPS 兼容重点：标题层级、表格边框、图片位置、代码块样式、批注和修订记录。
