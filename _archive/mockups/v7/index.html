#

Webpack利用

##

1. webpack简介

Webpack是一个前端模块打包工具。它可以将多个模块按照依赖关系进行静态分析，并生成一个或多个可部署文件。团队可以在此页用 Markdown 编写正文、代码、表格和图示，同时保留多人协作、评论锚点与版本回溯。

##

2. 利用工具

协作提示

输入 / 可插入 Markdown 块、任务清单、在线表格、流程图和代码块；右上角版本入口会把每次修改同步到项目变更时间线。

```bash
npm run build
npx source-map-explorer dist/main.js
```
Markdown块协作行为导出兼容
标题 / 大纲自动生成右侧目录docx、pdf、md
代码块支持评论和复制保留语言标记
表格支持多人编辑与变更记录兼容 Office / WPS
##

3. 参考文献
